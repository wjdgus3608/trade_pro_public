## 개발과정
1. 로직 구상
  * 하락장(종목의 60%이상 하락장) 일시 6시간 거래중지
  * 거래량 증가시 매수
  * X% 상승시 익절 
  * Y% 하락시 손절
  * 모든 코인 하루안에 청산
  * 거래량 상위 TOP X개만 매수
  * 1분봉으로 보기(스켈핑)
  * 3연속 음봉(X틱이상)일시 매수
  * 음봉+음봉이상의 양봉일시(X틱이상) 매수
3. 로직 구현

## 제한사항
  - 주문 요청 외 API는 초당 30회, 분당 900회 제한 (초당 15회정도 가능)
  - 주문 요청은 초당 8회, 분당 200회 제한 (초당 3회정도 가능)
